# Computer Networks and Routing Assignment (CNRoutingAssignment)

This repository contains the completed network configuration files for the routing lab assignment.

## 📁 Repository Contents
* **Task1_BasicConfig.pkt** - Basic router configurations including hostname changes and console security port password setups.
* **Task2_StaticRouting.pkt** - Triangular network layout connecting Computing, Business, and Science networks using manually mapped static routes.
* **Task3_RIP.pkt** - Linear chain topology utilizing dynamic RIPv2 routing protocols to advertise network links.
* **Task3_EIGRP.pkt** - The same linear chain layout optimized with EIGRP Autonomous System 10 dynamic routing.

## 🚀 How to Run
1. Download and install Cisco Packet Tracer.
2. Clone or download this repository locally.
3. Open any of the `.pkt` configuration files to view the live topology layouts and CLI parameters.
